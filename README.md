Library Management System:
A simple command-line application to manage your personal book collection.
Features:
Add books with details (title, author, year, genre, read status)
Remove books from your collection
Search for books by title
Display all books in your library
View statistics (total books and percentage read)
Simple menu-driven interface
When you run the program, you'll see a menu with 6 options:
Add a book - Enter book details to add to your library
Remove a book - Remove a book by entering its title and author
Search for a book - Find a book by its title
Display all books - View your entire collection
Display statistics - See total books and percentage you've read
Exit - Close the program
After each action, you can choose to perform another action or exit.
Example
Welcome to Library!
1. Add a book
2. Remove a book
3. Search for a book
4. Display all books
5. Display statistics
6. Exit
Please provide your choice: 1
Enter the book title: 1984
Enter the author name: George Orwell
Enter the publication year: 1949
Enter the genre: Dystopian
Have you read the book? (True/False): True
Note :
Books are stored in memory only. When you exit the program, all data is lost.
